About DREAMT

DREAMT consists of data and baselines for four open-ended, independent challenge problems designed for students learning about machine translation.

  1. The Alignment challenge: given a sentence-aligned text, produce word alignments.
  2. The Decoding challenge: given a translation model and some sentences, produce translations.
  3. The Reranking challenge: choose the best translation of a sentence from a list.
  4. The Evaluation challenge: given the output of several translation systems, rank them by accuracy.

Using DREAMT

To learn more about the design and use of DREAMT, read our paper, Learning to Translate with Products of Novices, which will be published in the Transactions of the Association for Computational Linguistics.

DREAMT is distributed under an MIT license, which means you're free to do whatever you want with it. If you use DREAMT in published academic work, please cite the above paper.

DREAMT has already been used in these courses. 1. The spring 2012 machine translation class at Johns Hopkins University. 2. The 2012 machine translation marathon, a week-long hackathon and summer school held annually in Europe. 3. The spring 2013 machine translation class at Carnegie Mellon University.
